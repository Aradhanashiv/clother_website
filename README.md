**E-Commerce Clothing Website
**    
A full-featured e-commerce web application for buying clothing online. 
This platform provides a seamless shopping experience with secure authentication, intuitive navigation, and efficient cart management.

**🚀 Features**  
🔐 Secure JWT Authentication    

User signup and login with JSON Web Tokens    
Protected routes and session handling  
🛒 Shopping Cart  
Add to cart functionality
Update quantity and remove items  
Persistent cart experience  
🔍 Search Functionality  
Quickly find products by name or category  
Dynamic filtering for better user experience  
👕 Category Sections    
Separate sections for:  
Men’s Clothing  
Women’s Clothing  
📦 Product Management  
View product details  
Organized product listings  
💻 Responsive Design  
Optimized for desktop, tablet, and mobile devices      

🛠️ Tech Stack  

**Frontend:**  
HTML, CSS, JavaScrit, React, Redux  
**Backend:**  
Node.js / Express.js  
**Database:**  
MongoDB  
**Authentication:**  
JSON Web Tokens (JWT)
